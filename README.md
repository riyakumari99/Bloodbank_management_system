## 🚀 Features

* 🔐 **User Authentication**

  * Secure login system with session management
  * Role-based access (Admin / User)

* 🧑‍🤝‍🧑 **Donor Management**

  * Register new donors with eligibility checks
  * Maintain donor history
  * Edit and delete donor records (Admin only)

* 🩸 **Blood Stock Management**

  * Automatic stock update on donation
  * Real-time availability tracking by blood group

* 📥 **Blood Request System**

  * Submit blood requests
  * Track request status (Pending / Completed / Rejected)

* ✅ **Blood Issuing**

  * Approve or reject requests
  * Automatic stock deduction on approval

* 📊 **Report Generation**

  * Download donor data as CSV report
